## 🙋‍♂️ About Me

<img align="right" src="https://media.giphy.com/media/f3iwJFOVOwuy7K6FFw/giphy.gif" width="250" />

Hey! I'm **Gaddam Rithvik** — a final-year **B.Tech CS (Data Science)** student at **Anurag University, Hyderabad**, passionate about **Machine Learning**, **Deep Learning**, and building AI-powered systems that solve real problems.

🧠 I've built:
- 🖼️ **CNN–LSTM Image Caption Generator** — InceptionV3 + LSTM trained on Flickr8k
- 📈 **Student Performance Predictor** — 91% accuracy with Random Forest, deployed on Flask
- 💬 **Real-Time Chat Application** — WebSockets with Socket.IO & Node.js

🏅 Certified by:
- ☁️ **AWS** — Cloud Foundations
- 🤖 **IBM** — Artificial Intelligence & Data Analysis with Python
- 🌐 **Great Learning** — Computer Networks

🚀 Eager to contribute to **AI**, **analytics**, and **data engineering** projects.

**Fun Facts:**
- 🏏 Key member of university cricket team — **1st place at HCA inter-college competition**
- 🧪 I treat every ML project like an experiment — hypothesis, test, iterate
- ⚡ My go-to debugging move: add another layer and see what happens
- 📊 I genuinely enjoy reading model evaluation metrics at 2 AM
- 🔥 Firm believer that clean code and clean data are equally important
- 🌟 From raw CSV to deployed prediction API — I do the full journey
